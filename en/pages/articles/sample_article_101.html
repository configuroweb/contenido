The Python interpreter and extensive standard library are freely available in source and binary code for most Python Web platforms, https://www.python.org/, and are freely redistributed. The same site also contains distributions and references to many free third-party Python modules, programs, tools, and additional documentation.


The Python interpreter is easily extensible with functions and data types implemented in C or C++ (or other languages ​​that allow them to be called from C). Python is also suitable as a language to understand modifiable applications.


This tutorial informally introduces the reader to the basic concepts and functionalities of the Python programming language and its system. It helps to have an accessible Python interpreter for hands-on experience, all examples are self-contained, allowing the tutorial to be used offline.


For a description of standard objects and modules, see The Python Standard Library. Reference to the Python language where a more formal definition of the language is provided. To write extensions in C or C++, read Extending and Embedding the Python Interpreter or the Python C API Reference Manual. There are several books that cover Python in detail.


This tutorial is not intended to be exhaustive or cover all language resources, not even the most used ones. Instead, it aims to present many two more notable resources and provide a clear idea of ​​the style and type of language that Python is. After reading it, you will be able to read and write Python modules and programs and will be ready to learn more about the various libraries and modules described in The Python Standard Library.
