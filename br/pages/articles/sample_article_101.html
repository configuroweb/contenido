O interpretador Python e a extensa biblioteca padrão estão disponíveis gratuitamente em código-fonte e binário para a maioria das plataformas da Web Python, https://www.python.org/, e são redistribuíveis gratuitamente. O mesmo site também contém distribuições e referências a muitos módulos, programas, ferramentas e documentação adicional gratuitos do Python de terceiros.


O interpretador Python é facilmente extensível com funções e tipos de dados implementados em C ou C++ (ou outras linguagens que permitem que sejam chamados a partir de C). Python também é adequado como linguagem para estender aplicativos modificáveis.


Este tutorial apresenta informalmente ao leitor os conceitos básicos e funcionalidades da linguagem de programação Python e seu sistema. Ajuda ter um interpretador Python acessível para experiência prática, todos os exemplos são autocontidos, permitindo que o tutorial seja usado offline.


Para obter uma descrição de objetos e módulos padrão, consulte The Python Standard Library. Referência da linguagem Python onde uma definição mais formal da linguagem é fornecida. Para escrever extensões em C ou C++, leia Extending and Embedding the Python Interpreter e o Python C API Reference Manual. Existem vários livros que cobrem o Python em detalhes.


Este tutorial não pretende ser exaustivo ou cobrir todos os recursos da linguagem, nem mesmo os mais usados. Em vez disso, visa apresentar muitos dos recursos mais notáveis ​​e fornecer uma ideia clara do estilo e tipo de linguagem que o Python é. Depois de lê-lo, você poderá ler e escrever módulos e programas Python e estará pronto para aprender mais sobre as várias bibliotecas e módulos descritos em The Python Standard Library.
