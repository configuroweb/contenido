Java é uma linguagem de programação e plataforma de computação para desenvolvimento de aplicativos. Foi lançado pela Sun Microsystem em 1995 e posteriormente adquirido pela Oracle Corporation. É uma das linguagens de programação mais utilizadas.


O que é a Plataforma Java?

A plataforma Java é uma coleção de programas que ajudam a desenvolver e executar programas escritos na linguagem de programação Java. A plataforma Java inclui um tempo de execução, um compilador e um conjunto de bibliotecas. JAVA é uma linguagem independente de plataforma. Não é específico para nenhum processador ou sistema operacional.
